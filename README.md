## 📋 Features

- **Adult Mosquito Surveillance** — Resting Collection & Human Landing Catch
- **Larval Surveillance** — Container inspection, dipping data
- **GPS auto-detect** with offline fallback
- **All India** State/UT + District dropdowns
- **WHO-standard indices** — PMHD, TMHD, HI, CI, BI, Pupal Index, Larval/Pupal Density
- **Export** — CSV, PDF report, WhatsApp, Email
- **Data persists** on reload (stored in device's localStorage)

---

## 🔄 Updating the App

1. Edit `index.html`
2. Bump the version number: `<meta name="est-version" content="2.0.X">`
3. Commit and push to GitHub
4. GitHub Pages publishes in ~1 minute
5. All users see the **"Update Available"** banner next time they open the app online
